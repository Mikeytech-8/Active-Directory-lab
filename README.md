# Active Directory Home Lab

## Objective

Build a Windows Server environment simulating a small company.

## Technologies

- Windows Server 2022
- Active Directory
- DNS
- DHCP
- Group Policy
- Windows 11 Client

## Architecture

1 Domain Controller
1 Windows Client

## Tasks Completed

- Installed AD DS
- Configured DNS
- Configured DHCP
- Created Users
- Created Security Groups
- Applied GPOs
- Joined Client to Domain

## Lessons Learned

- DNS troubleshooting
- GPO inheritance
- User permissions
